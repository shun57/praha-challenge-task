## 課題2

### application/x-www-form-urlencodedとapplication/json

- application/x-www-form-urlencoded
  a=1&b=1のようなデータがパーセントエンコーディングされて送信される。
- application/json
  {"a":1,"b":2} のようなJSON形式のデータが送信される
